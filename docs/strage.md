# Strage

## Volume
- Volume は予め用意されたボリュームをマニュフェストに直接指定することで利用可能となる．
- そのため，Kubernetes 上で新規にボリュームを作成したり，既存のボリュームを削除することは出来ない．
- 例えば Node 上の任意のパスをマウントすること出来たりする．
